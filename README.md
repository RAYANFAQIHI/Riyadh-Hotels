# Riyadh Hotels Analysis 🌙


A data analysis project on **Riyadh hotels** using **Python** and **Power BI**, presented with a Ramadan-inspired dashboard design.

## Tools
- Python
- Pandas
- NumPy
- Power BI
- Jupyter Notebook

## Project Steps
- Data cleaning and preprocessing
- Handling missing values
- Removing duplicates
- Price and rating analysis
- Comparing booking sources
- Building an interactive dashboard

## Dashboard Highlights
- Total hotels
- Average price
- Average rating
- Average discount
- Free cancellation distribution
- Top hotels by price
- Average price by booking source
